# ショコラの記憶  
## サービス概要
チョコレートの思い出を、未来の自分へ残すためのアプリケーションです。  
購入したチョコレートの写真や味の評価を記録できます。  
次回購入時の参考にでき、チョコレートの記憶を振り返ることもできます。
  
  
## 制作背景
チョコレートと一言で言っても、ボンボンショコラ・トリュフ・生チョコ・焼菓子・ケーキなど様々な種類やフレーバーが存在します。  
「このブランドのチョコはどんなものを食べたことがあったかな？」「ここのは甘めだったかな？」「去年友人にどこのどんなものを贈ったかな？」というような問題を解決するために生まれました。  

ターゲットはチョコレート好きの20〜40代の女性です。  
バレンタイン催事の時期に自分用にもチョコレートをたくさん購入していた、過去の自分にも届けたいアプリケーションです。
  

## アピールポイント ・ 差別化ポイント
- 自分の食べた記録だけでなく、贈答品についても記録を残しておけます。
- Webアプリケーション（インストール不要）なので、記録が増えてもストレージ容量の少ない端末でも続けられます。  

チョコレートの食べ比べを楽しんでいる方、大切な人へ贈ったチョコレートを記録として残しておきたい方々の一助になれば幸いです。
  

## MVP機能
#### 認証機能
- ログイン
#### 記録機能
- 登録（写真1枚、実食日、味評価、メモ）
- 一覧表示
- 詳細表示
- 編集
- 削除
  

## 本リリース追加予定機能
- 登録機能拡充
  - 写真複数枚、★総合評価、カテゴリ（チョコ/ケーキ/焼菓子など）
  - 贈答用記録（写真、宛名、贈答日、メーカー名、メモ）
- 検索/フィルタ表示（ブランド名、期間指定、カテゴリなどで絞り込み）
- 詳細画面のシェア機能
- トップページにランダムで労わる一言を表示
- UIの改善
  

### 技術予定スタック
|要素|技術|選定理由|
|-|-|-|
|バックエンド|Ruby on Rails|MVC構造が明確で迅速なWebアプリを開発できるため|
|データベース|PostgreSQL|Railsとの相性が良く、リレーショナルデータ管理に適しているため|
|UIデザイン|Tailwind CSS|ユーティリティファーストで効率的にデザインを構築できるため|
|認証|Devise|安全な認証機能を効率的に実装できるため|
|画像管理|Cloudinary|画像を外部保存でき、サーバー負荷を軽減できるため|
|開発環境|Docker|環境統一で将来的な保守性を高めるため|
